# Student OS - Chrome Extension

This extension gives the Student OS dashboard the ability to securely fetch your syllabus and grades directly from the university Moodle and Portal. 

Without this extension, those advanced AI and fetching features on the dashboard will not work.

## How to Install (Developer Mode)

Since this extension is not published on the Chrome Web Store, you need to load it manually. It takes 10 seconds:

1. **Extract this folder:** If you received this as a `.zip` file, right-click it and select "Extract All..." so it becomes a normal folder.
2. **Open Chrome Extensions:** Open Google Chrome, type `chrome://extensions/` into the URL address bar, and press Enter.
3. **Turn on Developer Mode:** In the top-right corner of the Extensions page, you will see a toggle switch labeled **"Developer mode"**. Click it to turn it ON.
4. **Load the Extension:** A new menu bar will appear at the top left. Click the button that says **"Load unpacked"**.
5. **Select the Folder:** A file browser will pop up. Navigate to the folder you extracted in Step 1, click on the folder so it is highlighted, and click **"Select Folder"**.

That's it! The extension is installed and running. You can now use the automated Moodle scraping features on the dashboard.

## Privacy & Security
This extension runs entirely locally on your machine. It does not send your grades, passwords, or data to any external servers or databases. It only acts as a secure bridge between your university portal and your private, local browser dashboard.
